taggal 30 kemarin saya membeli domain dengan alamat irfanywz.web.id

saya membelinya ditempat yang biasa saya beli, dengan harga tahun pertama yaitu sebesar 10rb, total bayar 13rb

setelah membelinya saya langsung setup domainnya agar terhubung dengan cloudflare

...

tepat tanggal 31 saya lanjutkan integrasi halaman github pages saya dengan domainnya

ternyata ada yang belum ditambahkan yaitu CNAME dengan name www dan tujuannya kehalaman github pagenya

...

alasan saya pindah dan membeli domain ini bukan tanpa alasan

- alasan yang pertama yaitu agar bisa diterima google adsense, karena sayang akunnya ada tapi websitenya tidak
- selain itu domain github.io tidak bisa di-index sitemapnya, pas saya cari dimesin pencaripun tidak ada yang muncul halaman web saya, hanya halaman utama saja yang ter-index
