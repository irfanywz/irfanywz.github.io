setelah sebelumnya saya membuat aplikasi vheer-cli, kali ini saya buat versi pollinations-cli (polli-cli)

polli-cli ini adalah ai yang bisa dipakai untuk generate gambar secara gratis


tujuan membuat banyak versi cli, yaitu karena 

aplikasi generate gambar ini bakal saya embed untuk aplikasi yang lebih besar.

mengembed lebih mudah daripada menulisnya sebagai kode berkali-kali, cukup panggil kode untuk mengeksekusi dan masukan parameter, maka selesai sudah

kalau ada error pada embednya saya tidak perlu mencompile ulang aplikasi utama, cukup aplikasi embednya saja
