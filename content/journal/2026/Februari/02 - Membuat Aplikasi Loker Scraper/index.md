pagi pagi saya bangun dengan ide yang sebelumnya pernah dipikirkan yaitu membuat aplikasi loker scraper

aplikasi ini saya buat dengan tujuan agar mempermudah dalam pencarian loker yang terupdate dari berbagai situs penyedia loker

aplikasi sudah berjalan dengan semestinya namun fiturnya masih terbatas, karena hanya bisa scraping setiap halaman

masih banyak yang harus dikembangkan seperti scraping loadmore, scraping virtual dom, scraping lewat api, dll

tapi saya rasa ini sudah lumayan untuk mencari sebuah loker terbaru

...

selain itu aplikasi loker scraper ini menggunakan stack react yang sudah mulai saya adaptasi untuk proyek kedepannya menggantikan vue

dengan tujuan untuk mengikuti perkembangan teknologi terkini
