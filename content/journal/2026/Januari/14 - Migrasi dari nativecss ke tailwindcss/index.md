sejak hari kemarin saya merebuild ulang template hugo web ini, agar sepenuhnya menggunakan tailwindcss4

kenapa hal ini saya lakukan ? ini dilakukan untuk kemudahan dan kesederhanaan kode

kalau pakai native kode terlalu panjang yang harus ditulis, tapi dengan library cukup dengan beberapa baris kode saja sudah bisa membuat hasil yang sama

jadi daripada menulis kode panjang-panjang, lebih baik sedikit kode tapi tujuannya sama saja. ini lebih effisien waktu dan kinerja

terlebih AI juga merekomendasikan penggunaan teknologi terbaru maka mau tidak mau saya harus beradaptasi untuk menyesuaikan

selain tailwindcss yang digunakan, saya juga menggunakan alpine js untuk pengelolaan state dan util javascript lainnya

bisa dibilang alpine js ini versi litenya dari vue js.

...

walau tampilan tidak sepenuhnya mirip seperti sebelumnya, ini sudah lebih dari cukup
