## Persyaratan Sistem

Pastikan komputer Anda memenuhi persyaratan sistem minimum untuk menjalankan Audio Extract Tool:

**Minimum**

- **OS**: Windows 10 (64-bit)
- **Prosesor**: 2.4 Ghz
- **Memori**: 8 GB RAM
- **Penyimpanan**: 2 GB ruang tersedia  
  
&nbsp; 
  
**Rekomendasi**

- **OS**: Windows 11
- **Prosesor**: 3.5 Ghz
- **Memori**: 32 GB RAM
- **Grafis**: (GTX 10xx, RTX 20xx, 30xx) Series
- **Penyimpanan**: 2 GB ruang tersedia

## Cara Instalasi

Ikuti langkah-langkah berikut untuk menginstal Audio Extract Tool:

1. **Ekstrak File**: Unduh file zip aplikasi yang telah diberikan setelah pembelian. Ekstrak seluruh isi file zip ke lokasi yang Anda inginkan di komputer Anda (misalnya, `D:\Apps\Audio Extract Tool`).
2. **Jalankan Aplikasi**: Masuk ke folder yang telah diekstrak, lalu cari dan jalankan file `Audio Extract Tool.exe`.


## Cara Aktivasi

1. **Buka Aplikasi**: Jalankan aplikasi `Audio Extract Tool.exe`.
2. **Salin Machine ID**: Pada jendela aktivasi, Anda akan melihat Tombol "Copy Machine ID". Salin ID ini.
3. **Kirim ke Admin**: Kirim "Machine ID" yang telah Anda salin ke admin melalui WhatsApp atau Telegram.
4. **Terima License Key**: Admin akan membalas dengan "License Key" yang sesuai dengan "Machine ID" Anda.
5. **Masukkan License Key**: Salin "License Key" yang Anda terima dari admin, lalu tempelkan ke kolom "License Key" di aplikasi.
6. **Aktivasi**: Klik tombol "Login". Jika berhasil, aplikasi akan terbuka dan siap digunakan.  


## Penggunaan

Setelah instalasi selesai, Anda dapat membuka aplikasi Audio Extract Tool dari menu Start atau shortcut di desktop Anda.

**1. Halaman Utama**  

Pada halaman utama, Anda akan melihat antarmuka yang sederhana dan mudah digunakan.  


![Halaman Utama Audio Extract Tool](tampilan-utama.avif)


**2. Memisahkan Vokal dan Instrumental**  

Untuk memisahkan vokal dan instrumental dari sebuah file audio atau video:

1. **Pilih File**: Klik tombol "Add" untuk memilih file audio atau video yang ingin Anda proses.
3. **Mulai Proses**: Klik tombol "Start" untuk memulai pemisahan.
4. **Selesai**: Setelah proses selesai, Anda akan menemukan dua file baru di lokasi output: satu untuk vokal dan satu untuk instrumental.

## FAQ

**Apakah Audio Extract Tool membutuhkan koneksi internet?**  

Tidak, Audio Extract Tool dapat berjalan sepenuhnya secara offline setelah aktivasi.

**Format file apa saja yang didukung?**  

Audio Extract Tool mendukung berbagai format audio dan video populer, termasuk MP3, WAV, FLAC, M4A, MP4, AVI, dan lainnya.

**Bisakah saya memproses banyak file sekaligus?**  

Ya, Audio Extract Tool mendukung pemrosesan multi-file, memungkinkan Anda menambahkan beberapa file ke antrean dan memprosesnya secara berurutan.

**Bagaimana jika saya mengalami masalah teknis?**  

Jika Anda mengalami masalah teknis, silakan masuk grup support telegram
