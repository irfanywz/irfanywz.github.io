## Pembelian

Anda dapat membeli lisensi Promise ETY melalui [halaman pembelian](/shop/?product=promise-ety). Setelah pembelian, Anda harus melakukan konfirmasi lewat whatsapp atau telegram untuk menerima tautan unduhan.

## Persyaratan Sistem

Pastikan komputer Anda memenuhi persyaratan sistem minimum untuk menjalankan Promise ETY:

**Minimum**

- **OS**: Windows 10 (64-bit)
- **Prosesor**: 2.4 Ghz
- **Memori**: 8 GB RAM
- **Penyimpanan**: 2 GB ruang tersedia  
  
&nbsp; 
  
**Rekomendasi**

- **OS**: Windows 11
- **Prosesor**: 3.5 Ghz
- **Memori**: 32 GB RAM
- **Grafis**: (GTX 10xx, RTX 20xx, 30xx) Series
- **Penyimpanan**: 2 GB ruang tersedia

## Cara Instalasi

Ikuti langkah-langkah berikut untuk menginstal Promise ETY:

1. **Ekstrak File**: Unduh file zip aplikasi yang telah diberikan setelah pembelian. Ekstrak seluruh isi file zip ke lokasi yang Anda inginkan di komputer Anda (misalnya, `D:\Apps\Promise ETY`).
2. **Jalankan Aplikasi**: Masuk ke folder yang telah diekstrak, lalu cari dan jalankan file `Promise ETY.exe`.


## Cara Aktivasi

1. **Buka Aplikasi**: Jalankan aplikasi `Promise ETY.exe`.
2. **Salin Machine ID**: Pada jendela aktivasi, Anda akan melihat id mesin yang bisa diselect untuk disalin.
3. **Kirim ke Admin**: Kirim "Machine ID" yang telah Anda salin ke admin melalui WhatsApp atau Telegram.
4. **Terima License Key**: Admin akan membalas dengan "License Key" yang sesuai dengan "Machine ID" Anda.
5. **Masukkan License Key**: Salin "License Key" yang Anda terima dari admin, lalu tempelkan ke aplikasi.
6. **Aktivasi**: Tekan "Enter" untuk melakukan aktivasi lisensi. Jika berhasil, aplikasi akan terbuka dan siap digunakan.
